# Winning Hooks

Only add hooks or hook patterns after repeated positive signals.

## Confirmed Winners

- None yet.

## Assumptions To Test

- Strong hooks usually say less and imply more.
- If a hook can be understood fully before the visual starts, it is probably too weak.
- The best hooks fit the clip instead of sitting on top of it.

## Hook Angles To Test

- Wait for the last second
- This sound is unreal
- You know this one wins
- No one expected that launch

## Promotion Guardrail

- One-off hits with no repeat signal
- Generic descriptive lines
- Hooks that worked only because the footage was exceptional
